### 前言

> 哈喽,大家好 我是 Hor 👨🏻‍💻。 这是一个蘑菇丁(工学云)签到|打卡的JS SDK🤖


提供了签到，打卡，获取实习计划等等API，可集成本SDK快速实现你需要的功能。

### 环境

- 开发工具： Vscode 前端人必备写码神器
- Node.js： 17.6.0 版本
- 浏览器：   Edge 非常友好的浏览器( Chrome 内核， Win11 自带，好用)
- 技术栈：   Vite + JavaScript + axios
- 包管理：   Yarn

## 目录结构

以下是系统的目录结构

```
├── static-files         // 静态文件
├── src
│    ├── api             // api请求
│    ├── axios           // 网络请求封装
│    ├── config          // 配置常量
│    ├── utils           // 加密工具
│    ├── main.js         // 暴露接口
└── vite.config.ts       // vite全局配置
```
